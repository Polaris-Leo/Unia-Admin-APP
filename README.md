# Unia 弹幕悬浮窗（Electron）

系统级透明置顶悬浮窗，可在使用其他软件时保持弹幕可见，并在悬浮窗内直接执行禁言操作。

## 功能

- **系统级透明** — 背景真正透明，可透过悬浮窗看到后面的程序画面；文字/用户名/表情图片保持 100% 不透明
- **始终置顶** — 使用 Electron `alwaysOnTop: 'screen-saver'`，置顶于所有应用之上；可通过 📌 按钮随时切换
- **实时弹幕** — 直接连接 Unia 后端 WebSocket，接收弹幕并渲染（含表情图片、舰长徽章）
- **点击禁言** — 点击用户名弹出禁言操作面板（本场 / 1小时 / 12小时 / 永久）
- **透明度调节** — 设置面板中拖动滑块调节背景透明度（10%–100%）
- **字体大小调节** — 11px–20px
- **明暗主题** — 点击 ☽/☀ 按钮切换，独立于浏览器主题
- **自动重连** — WebSocket 断开后 3 秒自动重连；Token 过期时提示重新登录
- **设置持久化** — 透明度、字体大小、主题、窗口位置/尺寸均保存到本地

## 环境要求

- **Node.js** v18+（与 Unia 主项目一致）
- **Windows 10/11**（macOS 理论可用，但未针对性测试）
- Unia 后端已启动并运行（默认 `http://localhost:3001`）

## 使用方法

### 方式一：直接运行（开发 / 自用）

```bash
cd electron-overlay
npm install        # 首次安装依赖（约 200MB）
npm start          # 启动悬浮窗
```

### 方式二：打包为 .exe 分发给他人

```bash
cd electron-overlay
npm install        # 安装依赖（含 electron-builder）
npm run dist       # 打包，输出到 dist/Unia-Overlay.exe
```

打包完成后 `dist/Unia-Overlay.exe` 即为独立便携版，**无需安装 Node.js 或任何运行环境**，直接双击运行。文件大小约 150–180 MB（内含 Chromium + Node.js 运行时）。

---

首次启动显示登录界面，填写：
- **服务器地址** — Unia 后端地址，如 `http://localhost:3001`
- **用户名 / 密码** — 与网页端登录相同

登录后 Token 自动保存，下次启动直接连接，无需重新登录。

## 界面说明

| 按钮 | 功能 |
|------|------|
| 📌  | 切换始终置顶（高亮=置顶中） |
| ☽/☀ | 切换深色/浅色主题 |
| ⚙   | 打开/关闭设置面板（透明度、字体大小） |
| ⏏   | 断开连接并重新登录 |
| ─   | 最小化窗口 |
| ✕   | 关闭窗口 |

拖动标题栏可移动窗口，拖动边缘可调整窗口大小。

## 与浏览器 PiP 悬浮窗的区别

| 特性 | 浏览器 PiP | Electron 悬浮窗 |
|------|-----------|----------------|
| 置顶范围 | 仅高于浏览器标签 | 高于所有应用（screen-saver 级） |
| 背景透明 | ✗ 不支持（Chrome 限制） | ✓ 系统级真透明 |
| 需要浏览器 | ✓ 需要打开 Chrome/Edge | ✗ 独立进程 |
| 安装要求 | 无 | 需要额外 `npm install` |
